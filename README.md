StoryPointYK — це автономний апаратний аудіоплеєр на базі мікроконтролера ESP32. Пристрій відтворює аудіофайли (MP3/WAV) з SD-карти при спрацьовуванні лазерного датчика відстані (ToF). Керування та конфігурація здійснюються через веб-інтерфейс.

---

## Основні можливості

- **Captive Portal та Автономність:** Автоматичне створення Wi-Fi точки доступу для початкового налаштування. Підтримка роботи в локальній Wi-Fi мережі або в автономному режимі.
- **Web-Dashboard:** Адаптивний веб-інтерфейс для керування пристроєм.
- **Файловий менеджер:** Завантаження та видалення аудіофайлів на SD-карту через браузер.
- **Налаштування відтворення:** Програмне регулювання гучності, налаштування дистанції спрацювання датчика, вибір режимів відтворення (послідовно, випадково, фіксований трек).
- **OTA Оновлення:** Можливість оновлення прошивки (.bin) "по повітрю" через веб-інтерфейс.
- **Діагностика:** Відображення логів та поточного статусу апаратних модулів у реальному часі.

## Апаратне забезпечення

Перелік необхідних компонентів:
1. ESP32 Development Board
2. MAX98357A (I2S Class-D Amplifier)
3. VL53L0X (Time-of-Flight I2C Sensor)
4. MicroSD Card Adapter (SPI)
5. Динамік 4Ω-8Ω (до 3 Вт)
6. Конденсатор 470 мкФ 16В (Електролітичний) — паралельно лініям живлення MAX98357A.
7. Конденсатор 0.1 мкФ / 104 (Керамічний) — паралельно лініям живлення MicroSD адаптера.

## Програмне забезпечення та Залежності

Для компіляції проекту необхідно встановити наступні бібліотеки в Arduino IDE:

1. **ESP8266Audio** (Earle F. Philhower, III) — декодування аудіо та вивід через I2S.
2. **ESPAsyncWebServer** (me-no-dev) — асинхронний веб-сервер.
3. **AsyncTCP** (me-no-dev) — залежність веб-сервера.
4. **Adafruit_VL53L0X** (Adafruit) — драйвер датчика відстані.

## Встановлення та Запуск

### 1. Компіляція та Прошивка
Для коректної роботи OTA-оновлень необхідно змінити схему розмітки пам'яті перед компіляцією.
В Arduino IDE перейдіть до `Tools` -> `Partition Scheme` та оберіть:
* **`Minimal SPIFFS (1.9MB APP with OTA/190KB SPIFFS)`** або **`Default 4MB with spiffs`**.

### 2. Налаштування мережі
1. Подайте живлення на пристрій. Буде створено відкриту Wi-Fi точку доступу **`StoryPointYK`**.
2. Підключіться до мережі та перейдіть за адресою `http://192.168.4.1` (або дочекайтеся переадресації Captive Portal).
3. Оберіть режим роботи, вкажіть дані доступу (SSID/Password) та облікові дані веб-інтерфейсу.
4. Натисніть "Зберегти та запустити". Пристрій буде перезавантажено.

### 3. Завантаження файлів
1. Відкрийте веб-інтерфейс за IP-адресою пристрою.
2. Перейдіть до розділу "Менеджер файлів".
3. Оберіть аудіофайл (.mp3 або .wav) та натисніть "Завантажити на пристрій".

## Вирішення проблем (Troubleshooting)

* **Відсутність звуку при коректному перемиканні треків у логах:** Перевірте фізичне підключення шини I2S (рекомендується пайка замість макетних з'єднань).
* **Переривання відтворення аудіо:** Переконайтеся у наявності блокувального конденсатора (0.1 мкФ) на лінії живлення SD-модуля.
* **Помилка завантаження OTA:** Перевірте правильність обраної `Partition Scheme` під час первинної прошивки через кабель.
